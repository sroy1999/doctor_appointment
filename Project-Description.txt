This project is about a web application through which the user can book appointments with doctors,
the doctor can see appointments and the admin can manage the doctors' profiles

There are three types of user logins:-
- User login
- Doctor login
- Admin login

The app is integrated with a payment gateway of razorpay

For backend:-

- Login to AWS console
- Setup a VPC and an EC2 instance
- Launch DocumentDB using the VPC and EC2 instance
- Configure Amazon S3 for storing the images
